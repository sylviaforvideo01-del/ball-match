# 球球消消乐 - 实时开发指南

## 快速开始

### 1. 解压文件
把 `ball-match-game.tar.gz` 解压到桌面，得到 `ball-match-game` 文件夹

### 2. 启动游戏
双击 `启动游戏.bat`

会弹出一个黑窗口，显示：
```
游戏地址: http://localhost:8080
```

### 3. 浏览器打开
用浏览器访问：http://localhost:8080

---

## 实时更新流程

每次我改完代码后：

1. **我会告诉你** "已更新，请刷新"
2. **你按 F5** 刷新浏览器
3. **立刻看到** 新版本

就这么简单。

## 常见问题

**Q: 黑窗口闪一下就没了？**
A: 可能没装 Python，去 https://python.org 下载安装

**Q: 端口被占用？**
A: 关闭其他程序，或重启电脑

**Q: 刷新后没变化？**
A: 按 Ctrl+F5 强制刷新，或清除浏览器缓存
